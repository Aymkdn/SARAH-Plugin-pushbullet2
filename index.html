Auteur: Aymeric (Twitter GitHub Blog)

Ce plugin permet d'envoyer des notifications Pushbullet.
Les sources sont disponibles sur GitHub: https://github.com/Aymkdn/SARAH-Plugin-pushbullet2

Vous devez configurer ce plugin en entrant votre access token disponible sur votre compte Pushbullet à l'adresse https://www.pushbullet.com/account. Il est également possible d'envoyer les notifications sur des appareils bien précis (et pas sur tous vos devices). Pour cela vous devez configurer le device_iden. Pour trouver le device_iden il faut vous rendre sur https://www.hurl.it/ :

  • Dans la zone URL entrer l'adresse https://api.pushbullet.com/v2/devices
  • Cliquer sur Add Header, puis, dans la première colonne entrer Access-Token, puis dans la deuxième colonne entrer votre_access_token
  • Cliquer sur Launch Request
Là vous allez avoir une réponse sous forme d'une chaine JSON. Il faudra repérer le device que vous voulez et récupérer son iden.

Utilisation du plugin

Le plugin s'appelle via d'autres plugins. Pour ce faire, les autres plugins doivent l'appeler avec la commande :
SARAH.trigger('pushbullet2', { "title" : "Titre de la notification", "content" : "Contenu du message" });

Il est également possible de l'appeler avec l'URL : http://127.0.0.1:8080/sarah/pushbullet2?title=mon%20titre&content=mon%20message

Remarque : il est possible de préciser le device qui sera notifié en ajoutant "device_iden" avec le bon iden (par défaut on utilise celui qui est indiqué dans la configuration, ou c'est envoyé à tous les appareils si aucun n'est spécifié).
